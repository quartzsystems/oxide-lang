## Type System

* [Bool](#bool)
* [Int](#int)
* [Float](#float)

### Bool
The `bool` represents a `true` or a `false` value.
```rust
let x: bool = true;
```

### Int
The `int` represents a integer number.
```rust
let y: int = 1;
```

### Float
the `float` represents a floating point number.
```rust
let z: float = 1.02;
